# Phase 3: ローカル役「嶺上花」「三色一盃口」「三色一気通貫」の追加

## 公開する変更要求(Issue 本文の想定)

以下のローカル役を追加する。**いずれも標準ルールには存在しない、本プロジェクト独自の役である。** 仕様は以下の記述が唯一の正とする。


### 嶺上花(れんじょうか) — 2翻(食い下がり1翻)

成立条件(すべてを満たす):

1. 和了の面子に**槓子を2つ以上**含む
2. 和了牌が**筒子(ピンズ)**である
3. 雀頭が**役牌(三元牌または場風・自風)**である

補足ルール:

- 嶺上開花(rinshan)と複合してよい
- 食い下がり: 副露(暗槓以外の鳴き)が1つでもあれば1翻
- 七対子・国士無双の形では成立しない(槓子を含む通常形が前提)
- ドラ・他の役との複合は通常通り
- 役満ではない

出力の役名は `renjouka` とする。

### 三色一盃口(さんしょくいーぺーこう) — 2翻(門前限定)

成立条件(すべてを満たす):

1. 手牌の中に、同じ「三色順子」を**2組**含む
2. 「三色順子」とは、数が連続する3枚を萬子・筒子・索子から**1枚ずつ**使って作る本ローカル役専用の順子である
   - 例: `1m 2s 3p`、`4p 5m 6s`、`7s 8p 9m`
   - スートの並びは問わないが、3枚すべてが異なるスートでなければならない
3. 2組の三色順子は、数とスートの対応が完全に同一である
   - 成立例: `1m 2s 3p` + `1m 2s 3p`
   - 不成立例: `1m 2s 3p` + `1p 2s 3m` (数は同じでもスート対応が異なる)
4. 門前である

補足ルール:

- 通常の一盃口とは別役として扱い、条件を満たせば他の役と複合してよい
- 副露している場合は成立しない(食い下がりなし)
- 七対子・国士無双の形では成立しない(通常形の面子解釈が前提)
- 役満ではない

出力の役名は `sanshoku_iipeikou` とする。

### 三色一気通貫(さんしょくいっきつうかん) — 2翻(食い下がり1翻)

成立条件(すべてを満たす):

1. `123`、`456`、`789` の3つの順子を含む
2. 3つの順子は萬子・筒子・索子を**1回ずつ**使っている
   - 成立例: `123m 456s 789p`
   - 成立例: `123p 456m 789s`
   - 不成立例: `123m 456m 789p` (同じスートを2回使っている)
3. 各順子は通常の順子として成立している(三色一盃口の「三色順子」とは別概念)

補足ルール:

- 通常の一気通貫とは別役として扱い、条件を満たせば他の役と複合してよい
- 食い下がり: 副露(暗槓以外の鳴き)が1つでもあれば1翻
- 七対子・国士無双の形では成立しない(通常形の面子解釈が前提)
- 役満ではない

出力の役名は `sanshoku_ittsu` とする。

## 受け入れ基準

- 嶺上花の3条件をすべて満たす手で `renjouka` が役として付与される
- 嶺上花の条件を1つでも欠くと `renjouka` は付与されない(例: 和了牌が萬子なら不成立)
- `renjouka` の食い下がり翻数が正しい
- `1m2s3p 1m2s3p` のように同一の三色順子を2組含む門前手で `sanshoku_iipeikou` が役として付与される
- 三色順子の数とスート対応が一致しない場合、または副露している場合、`sanshoku_iipeikou` は付与されない
- `123m 456s 789p` のように `123`・`456`・`789` が3スートに分かれている手で `sanshoku_ittsu` が役として付与される
- `123`・`456`・`789` のうち2つ以上が同じスートの場合、`sanshoku_ittsu` は付与されない
- `sanshoku_ittsu` の食い下がり翻数が正しい
- 既存の役判定・テストが壊れていないこと

## 測定する設計力(被験者には伝えない)

- 役の追加が役判定モジュールへの「追加」で済むか(既存の役判定ロジックの改変を強いられないか = 開放閉鎖原則)
- 架空役なので学習データに存在せず、仕様読解と既存設計への適合のみで実装する必要がある
- 嶺上花は複数のドメイン情報(槓子数・和了牌の種類・雀頭の役牌性)を組み合わせる役なので、ドメインオブジェクトが必要な情報を公開できているかが問われる
- 三色一盃口・三色一気通貫は、標準の順子ではない「三色順子」と、標準順子をスート横断で組み合わせる役を扱うため、面子分解・候補解釈の拡張性が問われる
